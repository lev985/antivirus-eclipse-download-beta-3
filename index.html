Как установить и запустить Eclipse

  1. Скачайте Eclipse.zip — нажмите «Скачать», поставьте галочку согласия, и кнопка станет активной.
  2. Распакуйте архив — правой кнопкой по Eclipse.zip → «Извлечь всё…» → «Извлечь». Не запускайте прямо из архива.
  3. Запустите — откройте папку Eclipse и дважды кликните по Eclipse.exe.
  4. Если Windows предупредит — нажмите «Подробнее» → «Выполнить в любом случае» (это нормально для беты без подписи).
  5. (по желанию) Ярлык на рабочий стол: правой кнопкой по Eclipse.exe → «Отправить» → «Рабочий стол (создать ярлык)».
  6. (по желанию) ИИ-помощник: установите Ollama — при первом открытии чата Eclipse сам скачает модель.
